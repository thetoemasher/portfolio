Projects

It's Showtime

React | Redux | Node | Express | Glamorous | PostgreSQL | DigitalOcean

It's Showtime is an app built to allow the user to search for movies theaters nearby and then view what movies are playing at that theater throughout the day. Then they can rearrange the movies based on whether or not they are interested in it, not interested in it, or have seen it already

Momstrong Move

React | Redux | CSS | Vimeo | Cron | Nodemailer | Apple Push Notifications

Momstrong's main purpose is to allow users to register (and adjust their subscriptions) for Momstrong Move, a fitness service. It also includes an admin side where the admin can go and create workout descriptions and link videos then schedule when those videos/workouts are to be viewable for the users.

React | Glamorous | Cron | Docker

I built this application for DevMountain as a way to easily set up pair programming for a class that will keep track of who a student has been paired with. This makes it so two students will not be paired together twice until they've gone through the whole class once.
